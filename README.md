## 实现功能

### 任务的添加 修改 删除 提前完成 检测系统时间与任务结束时间比较自动完成

### 任务组的添加 修改 删除(修改 删除需长按任务组那一栏)

### 任务的通知推送
在APP没被完全Kill时 APP会以两种方式推送通知
第一种是任务开始时间以及过了 但未到结束时间 此时立刻推送通知
第二种是任务开始时间未到 则会定时在任务开始时 推送通知
通知可根据符合条件任务 推送多个 并同时存在
